ISTRUZIONI PER CREARE UN FILE WORD DI TEST:

1. Apri Microsoft Word (o Google Docs)

2. Scrivi questo contenuto semplice:
   ---
   Test Document Upload
   
   Questo è un documento Word di test per verificare la funzionalità di upload.
   
   Contenuto:
   - Testo semplice
   - Nessuna immagine
   - Nessuna tabella
   - Nessuna macro
   
   Data: 2025-01-20
   Scopo: Test upload
   ---

3. Salva come:
   - Nome: "test-word-simple"
   - Formato: .docx (Word Document)
   - NON .doc (Word 97-2003)

4. Carica questo file nell'applicazione

5. Dovrebbe caricarsi in pochi secondi (non minuti)

Se anche questo file semplice ha problemi, il problema potrebbe essere:
- Versione di Word
- Impostazioni di salvataggio
- Metadati automatici di Word

ALTERNATIVA:
- Usa Google Docs
- Scrivi lo stesso contenuto
- Scarica come .docx
- Prova a caricare quello

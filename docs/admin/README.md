## Admin Features

### Super Admin
- Access to `/admin/usage` - View usage for all users
- Full system access
- User management (all operations)
- Billing management

### Admin
- Access to `/admin/dashboard` - System overview
- User management (view, edit roles/status)
- Billing overview
